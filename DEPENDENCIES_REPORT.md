# 资料与模块说明清单

## 本文件用途

用于帮助业务侧快速了解当前仓库中的主要说明资料分别在讲什么，方便查找对应内容。

## 主要资料

- README.md：整体业务使用说明。
- Instruction.md：业务范围、角色和模块归属说明。
- SETUP_GUIDE.md：首次使用系统的上手指南。
- QUICK_REFERENCE.md：高频操作速查。
- DELIVERY_SUMMARY.md：本次交付摘要。
- DELIVERY_COMPLETE.md：已交付内容说明。
- PROJECT_STATUS.md：当前业务状态说明。
- TECHNICAL_VERIFICATION.md：业务验收记录。
- QA_REGRESSION.md：回归检查清单。
- AI_TUIGKE_QUICKSTART.md：AI 推客快速上手。
- AI_TUIGKE_OPTIMIZATION_REPORT.md：AI 推客优化说明。
- AI_TUIGKE_MERGE_PLAN.md：AI 推客整合计划。

## 当前建议阅读顺序

1. 先看 README.md 了解整体系统。
2. 再看 SETUP_GUIDE.md 和 QUICK_REFERENCE.md 熟悉操作。
3. 按模块需要查看对应专项说明。
4. 验收时查看 DELIVERY_SUMMARY.md、TECHNICAL_VERIFICATION.md 和 QA_REGRESSION.md。
