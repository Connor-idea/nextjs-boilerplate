## 📞 快速支持

### 常见问题解决

**Q: 启动慢？**
A: Vite 首次启动会较慢，后续会更快。已启用缓存。

**Q: 样式错乱？**
A: 清除浏览器缓存或运行 `npm run dev`

**Q: 图标不显示？**
A: 确认 lucide-react 已安装，检查 import 语句

**Q: 端口被占用？**
A: 修改 vite.config.js 中的 port 配置

## 🎓 开发指南

### 添加新页面
1. 在 `src/modules/` 下创建新文件
2. 编写组件逻辑
3. 在 `App.jsx` 中添加路由

### 添加新样式
1. 使用 Tailwind 的实用类
2. 或在 `src/index.css` 中添加自定义 CSS

### 添加新图标
1. 从 lucide-react 导入
2. 使用与其他图标相同的方式
