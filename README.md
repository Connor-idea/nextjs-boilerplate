# CRM 业务使用说明

## 系统定位

本系统用于统一处理销售、管理和财务相关工作，帮助不同角色在同一个工作台内完成线索跟进、团队管理、提成查看、工作报告和财务核对等日常事项。

## 适用角色

- 超级管理员：查看全部模块与全部业务视图。
- 销售总监：查看团队整体进展、督促规则、提成管理和全员报告。
- 销售组长：查看本组成员、本组业绩、本组报告和本组提成。
- 销售专员：处理个人客户、个人线索、个人日报和个人提成预估。
- 财务：处理账单、结算、对账和提成发放状态。

## 当前主要模块

- 首页：查看数据概览、我的工作台、团队作战图、业绩目标、订单关联、分成配置。
- 员工管理：查看全部员工、直属员工、销售画像。
- 渠道管理：查看渠道信息、负责人和合作状态。
- 客户管理：查看客户列表、客户阶段、跟进状态。
- 销售线索：处理线索分配、AI 推客、客户跟进。
- 工作报告：提交日报、查看历史、查看周报汇总。
- 财务管理：处理供应商对账、提成管理、员工账单、结算订单等。
- 系统设置：维护督促规则配置。
- 个人中心：查看个人资料、消息和收款信息。

## 常用操作

### 1. 切换账号角色

在页面右上角点击当前账号，可切换到不同角色视角，查看该角色对应的菜单、数据范围和操作权限。

### 2. 进入首页工作台

- 销售专员可通过“我的工作台”查看今日待办、客户概况、逾期提醒和提成预估。
- 销售组长可在“我的工作台”基础上进入“团队作战图”。
- 销售总监和超级管理员可直接查看团队整体进展与预警。

### 3. 处理线索与客户

- 在线索管理中筛选、分配和跟进线索。
- 在 AI 推客中查看推荐客户、记录跟进结论。
- 在客户管理中查看客户当前状态和重点事项。

### 4. 查看团队与员工情况

- 在“全部员工”中查看全员信息。
- 在“直属员工”中查看本组或直属成员情况。
- 在“销售画像”中查看成员表现、负载和跟进结果。

### 5. 处理财务事项

- 在“供应商对账款”中核对账单和状态。
- 在“提成管理”中查看规则、月度汇总和发放状态。
- 在其他财务子页中查看结算、校正与账单信息。

### 6. 提交工作报告

- 在“AI日报/周报”中输入今日工作内容。
- 生成日报预览后可确认提交。
- 组长和总监可查看团队范围内的历史日报与周报汇总。

## 当前业务重点

- 销售角色以待办处理、客户推进、线索跟进为主。
- 管理角色以团队进度、督促规则、预警处理为主。
- 财务角色以账单核对、提成审核、结算处理为主。

## 使用提醒

- 页面右上角账号切换会改变菜单可见范围。
- 不同角色看到的数据范围不同，这是正常权限表现。
- 提成、日报、督促规则等页面当前已具备完整原型展示，可继续按业务规则补充实际数据。
